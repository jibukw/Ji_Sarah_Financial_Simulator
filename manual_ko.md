# Ji & Sarah 학자금 대출 상환 시뮬레이터 — 사용 매뉴얼

---

## 이 시뮬레이터는 뭐야?

Ji와 Sarah가 결혼 후 일본에서 함께 살면서 Sarah의 미국 학자금 대출($169,907)을 갚아나가는 과정을 시뮬레이션하는 도구입니다.

수입, 지출, 취업 상태, 출산, 이직 등 **매년 달라지는 상황**을 직접 입력하면, 대출이 언제 끝나는지, 이자를 얼마나 내는지, 저축은 얼마나 쌓이는지 자동으로 계산해줍니다.

**한국어/영어 전환**: 오른쪽 상단의 EN/한국어 버튼으로 언어를 바꿀 수 있습니다.

---

## 화면 구성

상단에 4개의 탭이 있습니다.

| 탭 | 무엇을 하는 곳인가 |
|---|---|
| 연도별 설정 | 매년의 수입·지출을 입력하고, 대출 설정을 조정하는 곳 |
| 결과 해설 | 시뮬레이션 결과를 연도별로 보고, "왜 이렇게 됐나" 설명을 보는 곳 |
| 비교 | 여러 시나리오(Sarah 취업/미취업, 아이 유무 등)를 한눈에 비교하는 곳 |
| 대출 상세 | 4개 대출의 금리·우선순위, Graduated 구조 설명, 전략 가이드를 보는 곳 |

## 상단 요약 숫자 (항상 보임)

화면 맨 위에 6개의 숫자가 있습니다. 설정을 바꾸면 실시간으로 바뀝니다.

| 항목 | 의미 |
|---|---|
| 완납 예상 | 현재 설정대로 하면 대출이 완전히 끝나는 시점 |
| 총 이자 | 대출 기간 동안 내는 이자 총액 (달러 + 엔화) |
| 누적 비상자금 | 시뮬레이션 종료 시점까지 쌓인 저축 총액 |
| 1년차 추가상환 가능 | 첫해에 추가상환할 수 있는 금액. "적자"면 추가상환 불가 |
| 초기 저축 | Sarah + Ji의 시작 저축 합계 |
| 총 상환액 | 원금 + 이자 전부 합친 총 납부 금액 |

## 탭 1: 연도별 설정

### 연도별 수입·지출 계획 테이블

이 테이블이 시뮬레이터의 핵심입니다.

**각 열의 의미:**

| 열 | 입력하는 것 | 예시 |
|---|---|---|
| Ji ¥ | Ji의 월 수입 (세후) | 300,000 |
| Sarah ¥ | Sarah의 월 수입 (세후). 미취업이면 0 | 0 또는 220,000 |
| 생활비 ¥ | 2인 월 생활비 (대출 납부 제외) | 300,000 |
| 육아 ¥ | 아이 관련 월 추가 비용. 없으면 0 | 0 또는 50,000 |
| 메모 | 그 해에 무슨 일이 있는지 메모 | "Sarah 건축 이직", "출산" |
| 여유분 | 자동 계산됨. 클릭하면 계산식 펼쳐짐 | ¥129,440 |

**통화 전환**: 테이블 오른쪽 위의 ¥ JPY / $ USD 버튼으로 입력 통화를 바꿀 수 있습니다. 달러로 입력하면 자동으로 엔화로 환산됩니다.

**콤마 표시**: 금액은 300,000처럼 콤마가 자동으로 붙습니다. 클릭하면 편집 모드로 바뀌고, Enter 또는 바깥 클릭으로 확정합니다.

**환산 금액**: 각 칸 아래의 작은 글씨가 반대 통화 환산액입니다. (¥300,000 아래에 $2,000 표시)

**연도 추가/삭제**: 하단의 [+ 추가] [- 삭제] 버튼으로 최대 15년까지 연도를 추가할 수 있습니다.

### 여유분이란?

여유분은 다음 계산으로 나옵니다:

```
여유분 = Ji수입 + Sarah수입 - 생활비 - 대출 최소납부 - 육아비 - 월저축
```

- **양수(+)**: 이 금액만큼 매달 추가상환 가능
- **적자(-)**: 추가상환 불가. 최소 납부만 진행
- **여유분 숫자를 클릭하면** 상세 계산식이 펼쳐집니다

**중요**: 적자여도 대출 상환이 멈추는 것은 아닙니다. 최소 납부는 계속 진행됩니다.

### 초기 저축 & 비상자금

| 항목 | 의미 | 기본값 |
|---|---|---|
| Sarah 저축 ($) | Sarah가 갖고 있는 저축 | $12,000 |
| Ji 저축 ($) | Ji가 갖고 있는 저축 | 직접 입력 |
| 월 저축 목표 | 매달 비상자금으로 빼두는 금액 | ¥30,000 |

월 저축 목표는 수입에서 먼저 빠지고, 나머지가 추가상환으로 갑니다. 적자인 해에는 저축도 자동으로 줄어듭니다.

### 대출 설정

**Graduated 2년마다 증가율** (가장 중요한 설정)

Graduated 상환 방식은 2년마다 최소 납부액이 올라갑니다. 슬라이더로 증가율을 설정합니다.

- 기본값: 15%
- 범위: 0% ~ 30%
- 0%로 설정하면 고정 납부로 계산
- 슬라이더 아래에 미리보기가 나옵니다: 현재 $1,004 → 2년후 $1,155 → 4년후 $1,328

**Auto Pay 금리 할인 (-0.25%)**

aidvantage에서 Auto Pay(자동이체)를 등록하면 금리가 0.25% 내려갑니다. ON/OFF 토글로 적용 여부를 선택합니다. 비교 탭에서 적용 전후 차이를 볼 수 있습니다.

**2027 플랜 종료 반영**

현재 Graduated Extended 플랜이 2027년 9~10월에 끝납니다. ON으로 켜면 플랜 종료 후의 새 납부액을 설정할 수 있습니다.

**목돈 일시 상환**

여유 자금이 생기면 한 번에 대출에 넣는 시뮬레이션입니다. 금액과 투입 시점(몇 개월차)을 설정합니다.

**상환 전략**

| 전략 | 설명 |
|---|---|
| 고금리 우선 (Avalanche) | 금리 높은 대출부터 갚음. 이자 절약 극대화. 추천. |
| 소액 우선 (Snowball) | 잔액 적은 대출부터 갚음. 심리적 성취감. |

## 탭 2: 결과 해설

### 연도별 상환 내역

각 연도별로 다음 정보를 보여줍니다:

| 열 | 의미 |
|---|---|
| 연초 잔액 | 그 해 시작할 때 남은 대출 |
| 이자 | 그 해 동안 낸 이자 |
| 원금 상환 | 그 해 동안 갚은 원금 |
| 연말 잔액 | 그 해 끝날 때 남은 대출 + 감소율(%) |
| 완납 | 그 해에 완납된 개별 대출 |
| 왜 이렇게 됐나 | 그 해의 상황 설명 |

**"왜 이렇게 됐나" 칸 읽는 법:**

맨 위에 컬러 뱃지가 전략 단계를 보여줍니다:
- 빨간색 **1단계: 버티기** — 적자 구간. 최소 납부만 진행
- 주황색 **2단계: 안정** — 소폭 양수. 소액 추가상환 시작
- 갈색 **3단계: 공격** — 여유 생김. 적극 추가상환
- 초록색 **4단계: 제거** — 본격 상환. 대출 하나씩 완납

그 아래에:
- Sarah의 수입 상태
- 적자/여유 상황과 추가상환 금액
- 최소 납부액 (Graduated 증가 반영)
- 저축 내역 (달러 + 엔화)
- 메모에 적은 이벤트

**모든 달러 금액 아래에 엔화 환산이 작은 글씨로 표시됩니다.**

### 전체 흐름 요약

테이블 아래에 텍스트로 전체 흐름을 한눈에 볼 수 있습니다. 각 연도의 잔액 변화와 누적 상환율(%)이 표시됩니다.

## 탭 3: 비교

### 시나리오별 비교

현재 설정을 기반으로 변수 하나만 바꿔서 비교합니다:

| 시나리오 | 바꾸는 것 |
|---|---|
| 현재 계획 | 연도별 설정 그대로 |
| Sarah 계속 미취업 | 모든 해 Sarah 수입 0 |
| Sarah 건축 즉시취업 | 모든 해 Sarah ¥300,000 |
| 아이 계획 없음 | 모든 해 육아비 0 |
| Auto Pay 적용시 | 금리 -0.25% (Auto Pay OFF일 때만 표시) |

각 시나리오의 완납 기간과 총 이자를 바 차트로 비교합니다.

### 핵심 인사이트

시나리오 비교 결과를 바탕으로 자동 생성되는 분석입니다. "Sarah가 건축 즉시취업하면 현재 대비 X년 단축, 이자 $X 절약" 같은 구체적 숫자를 보여줍니다.

## 탭 4: 대출 상세

### 대출별 현재 상태

4개 대출의 원금, 금리, 월 이자, 우선순위를 보여줍니다. Auto Pay가 켜져 있으면 할인된 금리도 같이 표시됩니다.

| 대출 | 원금 | 금리 | 우선순위 |
|---|---|---|---|
| 1-06 | $33,142 | 7.08% | 최우선 |
| 1-03 | $49,079 | 7.00% | 2순위 |
| 1-01 | $43,393 | 6.84% | 3순위 |
| 1-02 | $44,292 | 6.31% | 4순위 |

### 시뮬레이션 종료 시 잔액

시뮬레이션이 끝났을 때 각 대출의 남은 잔액 또는 "완납" 표시.

### Graduated 상환 구조 이해

Graduated가 왜 전체적으로 더 비싼지, 이걸 이기는 방법 4가지를 설명합니다:

1. 최소 납부 이상을 내야 기간이 줄어듦
2. 추가 상환분은 원금으로 직행
3. 고금리 대출부터 제거 (Avalanche)
4. 목돈은 나눠서가 아니라 한 번에, 빨리

### 전략 4단계 가이드

현재 상황에 맞는 전략 단계를 안내합니다:

- **1단계 (버티기)**: 수입 불안정할 때. 최소 납부 유지, 현금 확보
- **2단계 (안정)**: 2인 소득 시작. 소액 추가상환, 비상자금 확보
- **3단계 (공격)**: 안정적 소득. 적극 추가상환 $500+/월
- **4단계 (제거)**: 여유 확보. 목돈 투입, 대출 하나씩 완납

## 자주 묻는 질문

**Q: 여유분이 마이너스인데 완납 예상이 나와요?**
A: 최소 납부는 적자여도 계속 진행됩니다. 여유분 마이너스는 "추가상환이 없다"는 뜻이지 "대출을 안 낸다"는 뜻이 아닙니다.

**Q: Graduated 증가율을 몇 %로 설정해야 하나요?**
A: 실제 증가율은 aidvantage에서 확인해야 합니다. 보통 10~20% 범위입니다. 모르면 15%가 현실적인 중간값입니다.

**Q: 적자인 해에도 저축이 쌓이나요?**
A: 아닙니다. 적자이면 저축도 0입니다. 여유가 있지만 저축 목표보다 적으면 가능한 만큼만 저축됩니다.

**Q: 목돈은 언제 넣는 게 좋아요?**
A: 빠를수록 좋습니다. 단, 비상자금 3~6개월분은 반드시 확보한 뒤에 넣으세요.

**Q: 어떤 대출부터 갚아야 하나요?**
A: Avalanche(고금리 우선) 전략이 이자를 가장 많이 아낍니다. 1-06 (7.08%) → 1-03 (7.00%) → 1-01 (6.84%) → 1-02 (6.31%) 순서입니다.
